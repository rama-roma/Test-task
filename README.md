# Mini CRM

A simple customer management app built with Next.js.

## Features

- ✅ Add clients (name + phone)
- ✅ View all clients
- ✅ Edit client info
- ✅ Delete clients
- ✅ Form validation

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## How to Use

1. **Add Client:** Enter name and phone, click "Add"
2. **Edit:** Click "Edit" button, change info, click "Save"
3. **Delete:** Click "Delete" button

## Validation

- **Name:** Letters only (no numbers or symbols)
- **Phone:** Numbers and +, -, (), spaces

## Tech

- Next.js 14
- React 18
- TypeScript
